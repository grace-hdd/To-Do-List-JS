# 📝 Simple To-Do List Web App

A clean, responsive, and intuitive To-Do List application built with vanilla JavaScript, HTML, and CSS. This app allows users to manage their daily tasks efficiently with local data persistence.

---

## 🚀 Features

* **Add Tasks:** Quickly add items using the "Add" button or by pressing the **Enter** key.
* **Strike-through Completion:** Click on any task to mark it as completed.
* **Delete Tasks:** Remove tasks instantly by clicking the "×" close icon.
* **Data Persistence:** Uses `localStorage` so your tasks remain saved even if you refresh the page or close the browser.

---

## 🛠️ Tech Stack

* **HTML5** - Structure of the application
* **CSS3** - Styling and responsive layout
* **JavaScript (ES6)** - DOM manipulation and LocalStorage logic
